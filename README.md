## Overview of CFD Simulation of Blood Flow in a Patient-Specific Aneurysm

The main goal of this project is to use CFD  to simulate and analyze blood flow in a patient-specific aneurysm geometry. The goal was to assess possible rupture risks by evaluating hemodynamic parameters like wall shear stress.

## Tools used
- Star CCM+
- Computational Fluid Dynamics (CFD)
- Post-processing and flow visualization

## Methodology
- Import patient-specific aneurysm geometry into Star CCM+.
- Create a computational mesh to achieve adequate flow resolution.
- Simulate steady-state laminar flow conditions for 2 velocity configurations (0.4 and 0.8 m/s), which can be adjusted as necessary.
- Post processing.

## Key Results
- Estimated wall shear stress distribution on the walls of the aneurysm.
- Identified areas of high stress concentration, which are prone to rupture.
- Visualized the fluid flow patterns and velocity contours.

## Engineering Insights
- Wall shear stress is an important parameter in the assessment of rupture risk in the aneurysm.
- Geometry plays an important role in the occurrence of separation regions.
- CFD can be used for patient-specific medical analysis.
